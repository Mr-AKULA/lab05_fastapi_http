# Лабораторная работа №5 — Интеграция CNN в FastAPI

Сверточная нейронная сеть (CNN) для классификации изображений CIFAR-10,
интегрированная в трёхуровневое FastAPI-приложение.

## API — Эндпоинты

### Классификатор `/classify`

#### `POST /classify/` — классифицировать изображение

**Параметры:**
- `file` — файл изображения (JPEG, PNG, BMP, WebP и др.)
- `top_k` — сколько классов вернуть (1..10, по умолчанию 3)

**Пример запроса через curl:**
```bash
curl -X POST "http://127.0.0.1:8001/classify/?top_k=3" \
     -F "file=@C:/path/to/image.jpg"
```

**Пример ответа:**
```json
{
  "filename": "cat.jpg",
  "top_k": 3,
  "predictions": [

    {"label": "deer", "probability": 0.0231}

}
```

## Архитектура нейронной сети

### CIFAR10CNN

```
Вход: (B, 3, 32, 32)  ← цветное изображение 32×32, батч B штук

Блок 1:  Conv(3→32, 3×3) → BN → ReLU
         Conv(32→32, 3×3) → BN → ReLU
         MaxPool(2×2) → Dropout(0.25)
         Выход: (B, 32, 16, 16)

Блок 2:  Conv(32→64, 3×3) → BN → ReLU
         Conv(64→64, 3×3) → BN → ReLU
         MaxPool(2×2) → Dropout(0.25)
         Выход: (B, 64, 8, 8)

Блок 3:  Conv(64→128, 3×3) → BN → ReLU
         MaxPool(2×2) → Dropout(0.25)
         Выход: (B, 128, 4, 4)

Flatten: (B, 2048)

FC:      Linear(2048 → 256) → ReLU → Dropout(0.25)
         Linear(256 → 10)

Выход:   логиты (B, 10)  ← Softmax применяется при инференсе отдельно
```

| Параметр           | Значение  |
|--------------------|-----------|
| Всего параметров   | 667 178   |
| Размер весов       | 2.55 MB   |
| Входной размер     | 32×32×3   |
| Число классов      | 10        |

## Ключевые технические решения

### Singleton в сервисе
`service/classifier.py` реализует паттерн Singleton через `__new__`:
модель загружается **один раз** при первом обращении и переиспользуется
во всех последующих запросах. Без этого ~2.55 MB весов перечитывались бы
с диска при каждом HTTP-запросе.

### Согласование цветовых пространств
OpenCV читает изображения в порядке **BGR**, а PyTorch/PIL работают в **RGB**.
При несоответствии каналов точность падает. Конвертация: `cv2.cvtColor(img, cv2.COLOR_BGR2RGB)`.

### model.eval() при инференсе
- **Dropout** переходит в режим прохождения всех нейронов (вероятность обнуления = 0)
- **BatchNorm** использует накопленные при обучении статистики (`running_mean`, `running_var`),
  а не статистики текущего батча

### torch.no_grad()
Отключает граф автодифференцирования при инференсе: экономит ~30-40% памяти
и ускоряет вычисления (не хранятся промежуточные тензоры для backward).

### Нормализация
Параметры нормализации `mean=(0.4914, 0.4822, 0.4465)` и `std=(0.2470, 0.2435, 0.2616)` —
это среднее и стандартное отклонение по каналам RGB всего обучающего набора CIFAR-10.
Используются как при обучении, так и при инференсе — значения **должны совпадать**.
